# 브이월드를 활용한 야적장 임대관리 시스템의 설계 및 구현

Design and Implementation of Lease Management System for Open-Air Storage Yard using V-World

국토교통부에서 제공하는 오픈 맵 브이월드의 Open GIS API를 활용하여 현장에서 운영 중인 야적장 임대 현황을 시각화하여 보여주고, 원하는 곳을 클릭 한 번으로 선택하여 임대할 수 있게 하는 웹 시스템을 설계 및 구현하였습니다.
